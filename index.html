SW Admin Assets API

This is the image API for SW Admin application. Images are served directly from this GitHub Pages site.

Endpoints

Base URL: https://raw.githubusercontent.com/[username]/sw-admin-assets/main/api/v1/

  • /avatars/[filename] - User avatars
  • /team-badges/[filename] - Team badges and logos
  • /thumbnails/[filename] - Default thumbnails

Usage

Images can be accessed directly via their URLs. For example:

https://raw.githubusercontent.com/[username]/sw-admin-assets/main/api/v1/avatars/TheRock.jpg
